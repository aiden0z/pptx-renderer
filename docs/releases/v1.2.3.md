# v1.2.3 Release Notes

Release date: 2026-07-01

## Summary

v1.2.3 is a backward-compatible patch release focused on connector marker fidelity
and compact numeric text wrapping.

The release improves Office-authored connector arrowheads in flipped or transformed
slides, including marker direction, size, and placement. It also keeps compact numeric
tokens such as `80%` together during browser text layout, avoiding environment-sensitive
line breaks caused by font fallback or narrow text boxes.

## Fixed

- Connector arrowheads now preserve PowerPoint-like direction and marker placement more
  accurately when connectors are flipped or transformed.
- Connector marker sizing now follows the effective stroke and marker metadata more
  closely, reducing visible differences from PowerPoint/PDF output.
- Compact numeric text such as `80%` now stays together instead of allowing the browser
  to split the number and percent sign across lines.
- Adjacent numeric/unit runs such as `80` + `%` are grouped together during rendering
  so multi-run authoring does not reintroduce the same wrapping issue.

## Migration Notes

No migration is required.
