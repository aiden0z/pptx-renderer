# v1.2.1 Release Notes

Release date: 2026-06-25

## Summary

v1.2.1 is a backward-compatible patch release focused on PowerPoint rendering fidelity
for flipped grouped content and compact charts.

The release improves cases where Office-authored decks use flipped groups containing
text, images, connectors, tables, or charts. It also reduces overly dense default value
axis ticks in small column charts, bringing compact chart output closer to PowerPoint.

## Fixed

- Grouped content with horizontal and vertical flips now mirrors child geometry more
  accurately while preserving PowerPoint-like text orientation.
- Picture rendering now handles flipped shape clipping together with image crop metadata
  without incorrectly mirroring the bitmap inside the clip.
- Line-like connectors inside flipped groups now preserve arrow direction and marker
  placement more closely.
- Table and chart frames inside flipped groups now mirror their frame positions without
  flipping table or chart contents.
- Compact column charts now avoid overly dense default value-axis ticks when the chart
  frame is small.

## Migration Notes

No migration is required.
