# v1.2.2 Release Notes

Release date: 2026-06-28

## Summary

v1.2.2 is a backward-compatible patch release focused on PowerPoint rendering fidelity
for Office-authored charts, gradients, picture effects, tables, text, shapes, and
SmartArt.

The release improves chart defaults and edge cases across axis density, tick
visibility, label sizing, legends, markers, blank data points, data tables, rich chart
titles, text shadows, plot-area backgrounds, and radar manual layouts. It also refines
path gradients, picture and shape effects, table spacing, text defaults, selected
preset shapes, and SmartArt cycle pie offsets.

## Added

- The E2E comparison review page now provides a clearer review workflow for manual
  PDF-vs-HTML inspection.

## Fixed

- Charts now better match PowerPoint defaults for axes, ticks, labels, legends,
  markers, blank data points, data tables, rich titles, text shadows, plot-area
  backgrounds, and radar manual layouts.
- Pie, doughnut, radar, scatter, bubble, horizontal bar, stacked, stock, and dense line
  charts now render closer to Office output in compact and edge-case layouts.
- Gradient fills now handle path backgrounds, themed path gradients, focus rectangles,
  pixel-space radial radii, and subpixel strokes more accurately.
- Picture and shape rendering now handles clipped fills, grayscale effects, inner
  shadows, soft edges, and scaled-down shadows more closely to PowerPoint.
- Table and text rendering now better preserves outer paragraph spacing, paragraph tab
  defaults, vertical WordArt, and Office-like font fallback behavior.
- Shape and SmartArt rendering now handles fillable brace presets, flowchart storage
  guides, and SmartArt cycle pie offsets more accurately.

## Migration Notes

No migration is required.
